Contents

Parasolid provides tools that aid the efficient integration of Parasolid into your application to support complex application-specific functionality. This volume describes the support for tracking, managing and manipulating information in your application code. It explains how to refer to data within your application and how to define and use attributes. It provides information on the mechanisms used to implement feature trees, undo functionality and investigate alternative methods of producing a part. Information is also provided on how to track model changes and saving model data externally.

This volume contains the following chapters:

    • Chapter 93, “Identifying Application Data” describes how to refer to data within your application
    • Chapter 94, “Attribute Definitions” describes how to define attributes in your application
    • Chapter 95, “Attributes” describes how to work with attributes
    • Chapter 96, “Partitions” describe how to organise and save entities in a modeling session
    • Chapter 97, “Rollback” describes how to return a Parasolid session back to an earlier state
    • Chapter 98, “Groups” explains how to create collections of associated entities for further processing by your application
    • Chapter 99, “Bulletin Board” describes how to follow model changes during a modeling operation
    • Chapter 100, “Archives” describes how to save data to an external storage.
    • Chapter 101, “Debug Functionality” describes the Parasolid debug functionality.
