Contents

This volume describes the support Parasolid provides during a modelling session. It explains how to safely work with multiple threads, how to improve performance when using multiple processors and how to acquire information about a modelling session.

This volume contains the following chapters:

    • Chapter 113, “Calling Parasolid From Multiple Threads” describes how to call Parasolid when your application uses multiple threads
    • Chapter 114, “Symmetric Multi-Processing In Parasolid” describes how to use SMP
    • Chapter 115, “Session Support” describes the characteristics of and how to receive information about a Parasolid modelling session.
    • Chapter 116, “Version Controls”, describes the two methods of version control supported by Parasolid.
    • Chapter 117, “Implementing Version Control In Applications”, describes how to implement version control in your application.
    • Chapter 118, “Update Switches”, describes the update switches available to certain Parasolid function for use in conjunction with Parasolid’s legacy version control mechanism.
